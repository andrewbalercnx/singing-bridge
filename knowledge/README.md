# knowledge/

This directory holds Tier-3 deep-reference documentation for the project.
Files here are read on demand by Claude Code and by council reviewers (the
Domain Expert reviewer consults this directory before flagging domain
findings).

Bootstrap Step 3 can seed this directory from uploaded reference documents.
Typical contents include: architecture overview, protocol/domain primer,
data-model reference, API reference, test patterns, and deployment notes.

See the "Knowledge Base" section of the project's `CLAUDE.md` for the full
tiered knowledge model (Tier 0 codegraph, Tier 1 always-loaded CLAUDE.md +
MEMORY.md, Tier 2 directory-scoped CLAUDE.md files, Tier 3 this directory,
Tier 4 external MCP).

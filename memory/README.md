# memory/

This directory is Claude Code auto-memory. Files here persist across
conversations and are automatically loaded into the session context. Use it
for project-level scratch state such as `human-review-mode` toggles and an
auto-generated `MEMORY.md` index. See the "Knowledge Base" section of
`CLAUDE.md` (written by the bootstrap step) for the tiered knowledge model
and how this directory fits into it.
